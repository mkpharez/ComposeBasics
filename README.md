# ComposeBasics

An Android Compose application.

## Building the Project

This project includes Gradle wrapper files for easy building without requiring a pre-installed Gradle.

### Prerequisites
- JDK 11 or higher
- Android SDK (with SDK Platform for API 34)

### Building with Gradle Wrapper

On Linux/Mac:
```bash
./gradlew build
```

On Windows:
```bash
gradlew.bat build
```

### Project Configuration
- **Gradle Version**: 8.0.2 (compatible with Android Gradle Plugin 8.1.2)
- **Android Gradle Plugin**: 8.1.2
- **Kotlin Version**: 1.9.10
- **Target SDK**: 34
- **Min SDK**: 27

### Running the App
```bash
./gradlew installDebug
```

Or use Android Studio to open the project and run it directly.
